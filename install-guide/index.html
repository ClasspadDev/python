Your Classpad is Python ready

Follow these simple steps to get Python running on your ClassPad II

1

Copy PythonExtra-hh2.bin to your calculator

Copy the PythonExtra-hh2.bin file to the root of your calculator's storage, next to the run.bin file. Also copy any games and apps you want to run to the appropriate folders.

2

Launch Hollyhock

Go to the Settings menu, click the System menu, and select "Hollyhock Launcher". Then select "PythonExtra" from the list and click "Run".

3

Enjoy Python

Find where you put the game or app on your calculator and run it. Start exploring the power of Python on your ClassPad II!

If you need help with the installation, please refer to our detailed installation guide or join our Discord community.
